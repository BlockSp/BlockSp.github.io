Detailed Description

template<typename T, int B, int D = 1>
struct BlockSp::preConditioners::IChol0< T, B, D >

Block Incomplete Cholesky(0) preconditioner.

Approximates Ax = b by factoring A via Cholesky decomposition LL^T without fill-in. No pivoting. Using lower triangular matrices. For symmetric positive definite (SPD) linear systems, which are the user's responsibility to provide.
