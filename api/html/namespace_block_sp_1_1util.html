Detailed Description

blocksp_utility.hpp contains several utility functions for the BlockSp/blitz data structures.

These include:

1) Basic arithmetic, timers, random number functions.
2) TinyVector/Matrix operations, array/matrix manipulations.
3) Random BlockSp matrices and solArrays.
4) BlockSp to full dense matrix, solArray to full dense vector, and reverse.
5) Scalar to Complex BlockSp/solArray converters.
5) Explicit matrix inverse calculator for very small matrices.
6) Hash function for TinyVectors.
