Detailed Description

template<typename T>
requires (!is_complex_v<T>)
struct BlockSp::dense::SymEigenFactor_Arr< T >

Symmetric eigenvalue factorization - for real, symmetric 2D blitz::Arrays
A = QDQ^T, A^-1 = QD^-1Q^T.


Regularizing: removes near zero eigen values. Used for inverting diagonals in SOR, stores A^-1 and D.
