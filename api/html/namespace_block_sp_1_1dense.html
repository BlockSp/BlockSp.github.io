Detailed Description

blocksp_dense_multiply.hpp contains the dense linear algebra multiplication functions used in BlockSp.

blocksp_dense_solvers.hpp contains the dense linear algebra solver functions used in BlockSp.

These include dense matrix-matrix, matrix-vector products, dense identity, and dense matrix transpose.

There are two versions of each:

  • one is compile-time optimized, for TinyMatrices/TinyVectors.
  • the other is for blitz::Arrays, the size of which is specified at run-time.


These include: an explicit small matrix solver, LU and Cholesky factorizations/solvers, symmetric and nonsymmetric eigenvalue factorizations/solvers.
There are two versions of each:

  • one is compile-time optimized, for TinyMatrices.
  • the other is for 2D blitz::Arrays, the size of which is specified at run-time. these are suffixed by _Arr.
    There is also an eigenvalue calculator for 2D blitz::Arrays (not a solver).

template<typename T, int N>
blitz::TinyVector< T, N > BlockSp::dense::small_solve ( const blitz::TinyMatrix< T, N, N > & A,
const blitz::TinyVector< T, N > & b )

Dense matrix factorizations/solvers.

Solve Ax = b for x. A small, 1x1, 2x2, and 3x3 matrices.

Four factorizations/solvers for dense square matrices. 1) LUFactor - LU decomposition/solver, for general matrices.
2) CholeskyFactor - Cholesky decomposition/solver, for real symmetric positive definite (SPD) matrices.
3) SymEigenFactor - Eigenvalue decomposition/solver, for real symmetric matrices.
4) NonSymEigenFactor - Eigenvalue decomposition/solver, for general matrices.
with _Arr versions for 2D blitz::Arrays.
Also,
small_solve - Fast expicit solver for small 1x1, 2x2, and 3x3 linear systems.
Eigenvalues_Arr - Eigenvalue calculator for general 2D blitz::Arrays.
Note: NonSymEigenFactor(_Arr) can be used as a robust solver for general dense matrices.
