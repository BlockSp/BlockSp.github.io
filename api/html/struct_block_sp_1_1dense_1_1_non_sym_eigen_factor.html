Detailed Description

template<typename T, int N>
struct BlockSp::dense::NonSymEigenFactor< T, N >

Nonsymmetric Eigenvalue factorization - for real or complex nonsymmetric NxN matrix:
Schur decomp A = QUQ^T, A^-1 = QU^-1Q^T.

Stores A^-1 and eigenvalues. U upper triangular with eigenvalues on diagonal.
Regularizing: removes near zero eigen values. May be used for inverting block diagonals in SOR.
Note: there may be imaginary eigenvalues, which gives a block triangular U (real Schur form). Block sizes are 1 if eig real, 2 if imaginary. For real matrix, blocks are complex conjugate pairs.
