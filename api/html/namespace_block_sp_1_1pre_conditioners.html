Detailed Description

blocksp_preconditioners.hpp defines BlockSp's preconditioners for the iterative solvers.

Includes:
NoPre - A struct for no preconditioning. Does nothing.
Diag - A block diagonal preconditioner.
SOR - A block successive over-relaxation preconditioner.
ILU0 - A block incomplete LU preconditioner with zero fill-in and no pivoting.
IChol0 - A block incomplete Cholesky preconditioner with zero fill-in and no pivoting.
Additionally, the function makeADInv creates a vector of inverse solvers for the block diagonals.
