7//Welcome to BlockSp!
8//A templated C++ library for block sparse linear algebra. Version 1.0
9//
10//See more at https://BlockSp.github.io
11//
12//This software package is designed to address linear algebra problems
13// where the majority of elements within the linear system are zero (sparse)
14// and the non-zero entries are dense matrices (blocks).
15// Block sparse linear systems can be found in, e.g., high-order
16// finite element methods (FEM) and discontinuous Galerkin methods (DG),
17// as well as a wide range of other problems.
18//
19//This header-only C++ library provides support for both multiplying and solving
20// block sparse linear systems (Ax = b). The library includes
21// containers for block sparse matrices and multi-dimension block vectors,
22// block sparse matrix-matrix and matrix-vector multiplication,
23// preconditioned iterative block sparse system solvers
24// including conjugate gradient (pCG) and pGMRES,
25// and several preconditioners including block SOR and ILU0.
26//
27//The data type, block size, and dimension are specified at compile-time
28// via templating, allowing for fast optimized stack-based operations.
29// Through templating, BlockSp handles both real and complex problems.
30// BlockSp requires C++20 or later.
31//
32//Additional dependencies
33// To run BlockSp, you must first include and link the following libraries:
34// MKL -- Intel's Math Kernel Library, used for the dense/block linear algebra.
35// https://www.intel.com/content/www/us/en/developer/tools/oneapi/onemkl-download.html
36// blitz++ -- A high-performance matrix, vector, and multi-dimensional array library.
37// https://github.com/blitzpp/blitz
38//
39////////////////////////////////////////////////////////////////////////////
40////////////////////////////////////////////////////////////////////////////
41//
42//Here is a brief overview of the key files, containers, and functions
43// of the BlockSp library:
44//
45//blocksp_containers.hpp
46// BlockSp -- A block sparse matrix in condensed row storage (CSR) format.
47// solArray -- A multi-dimensioned block vector.
48// the file also includes the dot product, infinity norm, and two norm of solArrays.
49//
50// The containers use the following template parameters, specified at compile time:
51// Data type T -- float, double, std::complex<float>, and std::complex<double>.
52// int B -- block size of the solArray.
53// int D -- dimension of the solArray. D = 1 is the default.
54// Note: the block size of a BlockSp matrix it typically B or B*D.
55// Setting B = D = 1 reduces the problem to regular sparse linear algebra.
56// Testing indicates that the code often runs fastest when B = 2 and D = 1, use when applicable.
57//
58//blocksp_multiply.hpp
59// bsmm -- Block sparse matrix-matrix multiplication.
60// bsmv -- Block sparse matrix-vector multiplication.
61// transpose -- Block sparse matrix transpose.
62//
63//blocksp_solvers.hpp
64// BlockSp provides the following iterative linear system solvers.
65// pCG -- Preconditioned block conjugate gradient.
66// pGMRES -- Preconditioned block GMRES.
67// _dr -- pGMRES with deflated restarting (in blocksp_deflated_restart.hpp).
68// arnoldi_eigenvalues -- Arnoldi eigenvalue solver.
69//
70//blocksp_preconditioners
71// The following functions precondition the linear system for the iterative solvers:
72// Diag -- Block diagonal preconditioner.
73// SOR -- Successive over-relaxation.
74// ILU0 -- Incomplete LU factorization with zero fill.
75// IChol0 -- Incomplete Cholesky factorization with zero fill.
76//
77//Additional files in the library include:
78// blocksp_dense_multiply.hpp -- Dense matrix-matrix and matrix-vector multiplication.
79// blocksp_dense_solvers.hpp -- Dense linear system solvers.
80// blocksp_include.hpp -- All of the include files for running BlockSp.
81// blocksp_lapack.hpp -- Templated access to the Lapack routines used by BlockSp.
82// blocksp_utility.hpp -- Useful utility functions.
83//
84// BlockSp Copyright © 2026, Luke P. Corcos Ph.D. (lpcorcos@gmail.com)
85//
