Detailed Description

template<typename T, int N>
requires (!is_complex_v<T>)
struct BlockSp::dense::SymEigenFactor< T, N >

Symmetric eigenvalue factorization - for real, symmetric NxN matrix:
A = QDQ^T, A^-1 = QD^-1Q^T.

Stores A^-1 and D.
Regularizing: removes near zero eigen values. May be used for inverting block diagonals in symmetric SOR.
